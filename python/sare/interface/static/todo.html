    Wall B — Grounding

    Every fact is a string. SARE-HX has never felt heat. Human concepts are grounded in sensorimotor experience — all SARE knowledge is hollow symbol manipulation.

    Wall C — Integration

Tier 3 — Research (In Progress) 7 items · actively building · ~85-88% projected on completion
Symbol Grounding Problem
How do symbols acquire meaning from sensorimotor experience? (Harnad, 1990). "fire" is stored as a string — but a human's concept of fire includes felt heat, seen light, heard crackle. Every SARE symbol is hollow. Current SOTA: embodied simulation + VLMs provide partial grounding, not full solution.
🔨 IN PROGRESSResearch
Genuine Open-Ended Creativity
How does a mind generate truly novel concepts — not by analogy to existing ones, but by genuinely new combination? HypothesisMaker generates "wolf is_a mammal" by structural analogy. That's pattern matching, not creativity. No consensus on the mechanism for genuine conceptual novelty.
🔨 IN PROGRESSResearch
Catastrophic Forgetting
Learning new tasks destroys performance on old ones. No complete solution exists. EWC, PackNet, progressive networks all provide partial mitigation but break down at scale or require task labels. SARE currently has no forgetting protection — adding a new domain could corrupt existing symbolic transforms.
🔨 IN PROGRESSResearch
Unified Consciousness / Global Workspace Integration
What integrates 50 modules into a single coherent agent? Global Workspace Theory (Baars) proposes a "workspace" broadcasts to all modules simultaneously. SARE's event_bus is a GWT analog — but without a unified attention bottleneck, there's no single coherent "I". Philosophically open question.
🔨 IN PROGRESSResearch
Sample-Efficient Causal Learning
Building accurate causal models from 3-5 examples, not 3,000. Causal Bayesian Networks with interventions work at small scale. SARE's WorldModel needs 50+ observations to promote a rule with confidence. Humans build causal models from single observations. No reliable engineering path at scale.
🔨 IN PROGRESSResearch
Meta-Meta-Learning (Learn New Ways of Learning)
MAML / Reptile learn to fine-tune faster — but only via gradient descent on existing architectures. SARE's AlgorithmSelector learns which strategy to use, but the set of strategies is fixed. True meta-meta-learning would invent new learning algorithms from scratch. Not yet achieved by any system.
🔨 IN PROGRESSResearch
Intrinsic Motivation in Open Worlds
Curiosity-driven RL (ICML, ICM) works in bounded game environments. SARE's HypothesisMaker generates hypotheses but doesn't have reward shaping that generalizes to truly unbounded domains. Intrinsic motivation that drives exploration when there's no external reward signal — open research problem.
